#Eh?

A challenge to write a programming language patterned after Canadian stereotypes.

###Prototype
```
a Canadian
  with toque
  with scarf
  with broom

  can curl
    if skip:
      say "Hurry!"
    eh?
    if lead:
      pass
    eh?
    say "How social the game..."
  eh?

  can say_aboot:
    say "What's it all aboot?"
  eh?
eh?
```
