#EH?

A language for learning to program in the beautiful Canadian dialect.

###Prototype
```
a Canadian
  with toque
  with scarf
  with broom

  can curl
    if skip:
      say "Hurry!"
    if lead
      say "That's too much ice"
    eh?
  eh?

  can say_aboot
    say "What's it all aboot?"
  eh?
eh?
```

called from command line
```bash
$ canada example.eh
```
